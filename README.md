# 📚 SQL Bookstore Project

## About
This project is a Bookstore Database built using PostgreSQL. It demonstrates SQL skills by analyzing book, customer, and order data.

## 🛠️ Tools Used
- PostgreSQL
- pgAdmin 4
- SQL
- Git
- GitHub

- Aggregate Functions
- JOINS
